# Security policy

This repository contains **educational** interview preparation material (markdown and static HTML). It is not a running application with a traditional attack surface.

## Reporting security issues

If you discover a **security problem** related to this project—for example, accidentally committed secrets, malicious links in content, or a vulnerability in published static site configuration—please report it responsibly:

- **Preferred:** Open a **private** security advisory on the GitHub repository (if enabled), or contact the maintainer via the profile/contact links in [`social.html`](social.html).

Include enough detail to reproduce or verify the issue (file path, commit, or URL).

## Scope

- **In scope:** Content integrity, PII or secrets in the repo, supply-chain concerns in build or CI configuration if added later.
- **Out of scope:** General debate about offensive security pedagogy; vulnerabilities in third-party sites linked from study notes (report those to the site owner).

Thank you for helping keep learners safe.
