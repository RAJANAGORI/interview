# Study Plan

This plan ties together the **topic library**, **role paths**, and **mastery framework** so preparation stays structured—not random browsing.

---

## Before you start

1. Run **[Baseline Diagnostic - Interview Preparation](Baseline%20Diagnostic%20-%20Interview%20Preparation.md)** (optional but recommended) — pick a path with evidence, not guesswork.
2. Read **[Content Mastery Framework](Content%20Mastery%20Framework.md)** (10 minutes) — understand the four learning layers and folder contract.
3. Skim **[Topic Syllabus Index](Topic%20Syllabus%20Index.md)** — note your tier (foundation vs product vs staff).
4. Pick **[Role-Based Study Paths](Role-Based%20Study%20Paths.md)** — Path A (Product Security), B (AppSec), C (Staff+), or D (21-day sprint).
5. Bookmark **[Interview Round Playbook](Interview%20Round%20Playbook.md)** and **[Topic Readiness Rubric - Timed Scoring](Topic%20Readiness%20Rubric%20-%20Timed%20Scoring.md)** for weekly drills and scoring.

---

## 12-week baseline (part-time)

| Week | Focus | Actions |
|------|--------|---------|
| **1–2** | Foundations + identity | Encryption vs Hashing, TLS, JWT, OAuth, Cookies, CORS/SOP — Comprehensive + Critical Clarification |
| **3–4** | Core web vulns | SQLi, XSS, CSRF, IDOR, SSRF — prioritize your weak spots |
| **5–6** | Design-time skills | Threat Modeling, Secure Code Review, Authorization vs Authentication |
| **7–8** | Cloud & platform | Cloud Security Architecture, IAM / Least Privilege, Secrets, Containers, IaC |
| **9–10** | Scale & operations | CI/CD + Supply Chain, Observability, IR (production + multi-team) |
| **11** | Staff themes | Risk metrics, Business logic abuse, GenAI security (as needed) |
| **12** | Capstone | Role-specific scenarios, mock interviews, Quick Reference sweeps |

Adjust pace: strong on web? Compress weeks 3–4; weak on cloud? extend 7–8.

## Daily templates

### 45-minute day

- 15 min: Previous topic Quick Reference  
- 25 min: New section in Comprehensive Guide OR one Interview Q&A block  
- 5 min: One question answered **out loud**

### 2-hour deep day

- 60 min: Comprehensive Guide (new topic)  
- 30 min: Critical Clarification + notes  
- 30 min: Interview Q&A (timed answers)

---

## Weekly review (Sunday or fixed slot)

- [ ] Spaced repetition: **3** Quick References from earlier weeks  
- [ ] One **mock question** from a random topic  
- [ ] Update **[Progress Tracker](%F0%9F%93%8A%20Interview%20Preparation%20Progress%20Tracker.md)** or dashboard in the web app  

---

## Using the interactive app

Open `interview/index.html` (or your deployed site). Topics load from **`interview/Config/topics.json`**. Mark modules complete as you finish **Comprehensive**, **Mastery Track** (where present), **Interview Questions**, **Critical Clarification**, and **Quick Reference**.

---

## When the interview is in 7 days

- Drop deep reading; run **Quick Reference** for all high-priority topics (see Role Paths).  
- Drill **Interview Questions** aloud.  
- Prepare **6–8** STAR stories (see Microsoft pack Mastery Track story template if applicable).  
- Sleep > cramming.

---

## After the interview

Archive notes; tag topics where you felt weak. Re-enter the syllabus at those topics with the **Mastery Framework** upgrade checklist.

Good luck — preparation should make you **calmer**, not more anxious.
