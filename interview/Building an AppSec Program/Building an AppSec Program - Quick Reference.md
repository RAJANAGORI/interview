# Building an AppSec Program — Quick Reference

## 30 / 60 / 90

| Window | Focus |
|--------|-------|
| **30d** | Inventory, relationships, secret scan, MFA |
| **60d** | Risk tiers, CI baseline, champions, threat models |
| **90d** | Pilot gate, pen test, metrics, roadmap |

Mission · Scope · Stakeholders · Metrics · Escalation

Design reviews · Consult SLAs · Champion sync · Monthly metrics

~1 AppSec : 50–150 devs (varies by automation/industry)
