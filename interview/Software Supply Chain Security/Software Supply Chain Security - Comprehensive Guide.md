# Software Supply Chain Security - Comprehensive Guide

## Scope
Protect dependencies, build steps, artifact registries, and runtime deployment integrity.

## Important frameworks
- SBOM for dependency visibility.
- SLSA for build and provenance maturity.
- In-toto/Sigstore for attestation and verification.

## Practical program
- Maintain asset inventory of repos, registries, and package managers.
- Enforce trusted sources and dependency pinning.
- Verify provenance at deploy-time, not only at build-time.
- Run dependency risk triage with exploitability context.
