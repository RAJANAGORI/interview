# Rapid Security Triage (Fast Checking) - Quick Reference

## 60-second definition
- High-speed signal triage to separate noise from actionable risk.

## High-signal indicators
- high inbound report volume
- alert fatigue
- duplicate finding churn

## Common failure patterns
- severity inflation
- missing reproducibility gate
- inconsistent triage rubric

## Control priorities
- triage rubric with confidence scores
- dedup and clustering
- time-boxed validation workflow

## 2-minute answer skeleton
- Definition + boundary
- Failure mechanism
- Impact chain
- Mitigation plan
- Verification criteria
