# Tool Quick Cards

## Burp Suite
- Use for manual web testing, repeater/intruder workflows, proxy analysis.
- Interview note: focus on methodology, not tool-only dependency.

## Semgrep
- Lightweight static rules for code anti-patterns and secure coding checks.
- Interview note: good for fast guardrails and CI integration.

## CodeQL
- Deep semantic code analysis, strong for variant hunting in large codebases.
- Interview note: great for custom query-driven AppSec programs.

## Trivy
- Container/image/dependency/secrets scanning in CI and registries.
- Interview note: pairs well with supply-chain controls.

## ScoutSuite
- Multi-cloud security posture and misconfiguration visibility.
- Interview note: posture snapshot, not a full threat model.

## BloodHound
- Graphs AD identity permissions and attack paths.
- Interview note: useful for both offensive simulation and defensive cleanup.

## Nuclei
- Template-driven scanning for known issue classes.
- Interview note: validate findings and reduce false positives.

## Prowler
- Cloud benchmark/compliance and security checks.
- Interview note: useful baseline for cloud hygiene pipelines.
