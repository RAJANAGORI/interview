# OSI Layer — Quick Reference

| L | Name | Remember |
|---|------|----------|
| 1 | Physical | Medium, bits |
| 2 | Data link | Frames, MAC, switches |
| 3 | Network | Packets, IP, routers |
| 4 | Transport | TCP/UDP, ports |
| 5 | Session | Sessions (often merged in practice) |
| 6 | Presentation | TLS often discussed here academically |
| 7 | Application | HTTP, DNS, SMTP |

**TCP/IP (4 layers):** Link · Internet · Transport · Application

**Security one-liner:** Lower layers = path & reachability; **L7** = most **app** vulns; **TLS** protects the wire, not business logic.
